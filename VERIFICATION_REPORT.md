## 五、管理命令

```bash
# 查看定时任务状态
launchctl list | grep arxiv

# 手动触发一次
launchctl start com.arxiv.scheduled-fetch

# 查看日志
tail -f ~/logs/arxiv-fetch.log

# 停止服务
launchctl unload ~/Library/LaunchAgents/com.arxiv.scheduled-fetch.plist

# 启动服务
launchctl load ~/Library/LaunchAgents/com.arxiv.scheduled-fetch.plist

# 手动执行抓取脚本
cd backend && source venv/bin/activate
python scheduled_fetch.py              # 抓取今天和昨天
python scheduled_fetch.py --date 2026-03-15  # 抓取特定日期
```
